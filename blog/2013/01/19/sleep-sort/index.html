Sleep Sort

Jan 19th, 2013

看到一个搞笑的shell脚本实现的排序算法sleep sort:

#!/bin/bash
function f() {
    sleep "$1"
    echo "$1"
}
while [ -n "$1" ]
do
    f "$1" &
    shift
done
wait

写了个python版本的sleep sort

import time
import thread


def timer(x):
    time.sleep(x)
    print x

if __name__ == '__main__':
    arr = map(int, raw_input('input:    ').split(' '))
    for x in arr:
        thread.start_new_thread(timer, (x,))
    time.sleep(max(arr)+1)
